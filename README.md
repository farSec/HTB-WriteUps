# HTB Writeups

This repository contains my Hack The Box machine writeups, focusing on methodology, enumeration, exploitation, and privilege escalation techniques.

## Methodology

Each writeup follows a structured approach:

* Reconnaissance (Nmap, service enumeration)
* Service analysis and vulnerability identification
* Exploitation
* Privilege escalation
* Lessons learned

## Notes

* Only retired machines are included in this repository
* Writeups are created for educational purposes
* Focus is on understanding vulnerabilities, not just exploiting them
